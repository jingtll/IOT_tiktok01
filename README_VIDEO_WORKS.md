# 视频作品标签栏实现指南

## 功能概述

这个实现提供了一个完整的标签栏视频展示系统，具有以下特点：

### 主要功能
1. **标签栏切换** - 支持多个分类标签（全部、教程、演示、案例）
2. **响应式视频网格** - 自动适应不同屏幕尺寸
3. **视频播放功能** - 点击视频可全屏播放
4. **美观的UI设计** - 现代化的卡片式布局

### 文件结构
```
src/
├── components/
│   ├── TabBar.vue         # 标签栏组件
│   ├── VideoGrid.vue      # 视频展示组件
│   └── VideoWorks.vue     # 主视频作品组件
└── App.vue               # 应用入口（已更新）
```

## 使用方法

### 1. 启动项目
```bash
cd vue/vue-demo
npm run dev
```

### 2. 访问地址
打开浏览器访问 `http://localhost:5174`

### 3. 功能说明

#### 标签栏操作
- 点击不同的标签可以切换到对应的视频分类
- 当前选中的标签会有特殊的视觉反馈

#### 视频播放
- 点击任意视频卡片会弹出全屏播放器
- 支持 YouTube 链接自动转换为嵌入格式
- 点击关闭按钮或模态框外部区域可退出播放

#### 响应式设计
- 在桌面端：显示多列视频网格
- 在移动端：自动调整为单列布局
- 视频缩略图支持悬停动画效果

## 自定义配置

### 修改标签内容
编辑 `src/components/VideoWorks.vue` 文件中的 `tabList` 数据：

```javascript
tabList: [
  { title: '我的分类', key: 'myCategory' },
]
```

### 添加视频数据
在同一个文件中修改 `videoData` 对象，为每个分类添加视频：

```javascript
videoData: {
  myCategory: [
    {
      id: 1,
      title: '自定义视频标题',
      description: '视频描述内容',
      thumbnail: 'https://example.com/thumbnail.jpg',
      url: 'https://www.youtube.com/watch?v=your-video-id',
      duration: '10:30',
      views: 1000
    }
  ]
}
```

### 样式定制
- `TabBar.vue` - 修改标签栏的外观和交互样式
- `VideoGrid.vue` - 调整视频卡片的布局和视觉效果

## 技术特性

### Vue 3 特性
- 使用 Composition API 风格
- 响应式数据绑定
- 组件间通信通过 props 和事件

### CSS 特性
- Grid 布局实现响应式网格
- Flexbox 用于对齐和布局
- CSS 过渡动画增强用户体验
- Media queries 实现移动端适配

## 故障排除

### 常见问题
1. **开发服务器无法启动** - 检查端口是否被占用
2. **组件未找到错误** - 确认所有组件文件都已正确创建
3. **样式不生效** - 检查 CSS 类名是否正确拼写

### 性能优化
- 对于大量视频，考虑实现虚拟滚动
- 图片懒加载以提高页面加载速度
- 缓存 API 响应减少网络请求

这个实现为你提供了一个完整的基础框架，你可以根据具体需求进行进一步的定制和扩展。
